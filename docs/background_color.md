# Background Color
There are a few different classes that you can add to your HTML elements in order to change their background color, one for each of the colors in the variable section:
- `cs-bg-is-primary`
- `cs-bg-is-secondary`
- `cs-bg-is-yellow`
- `cs-bg-is-red`
- `cs-bg-is-orange`
- `cs-bg-is-purple`
- `cs-bg-is-grey`
- `cs-bg-is-black`
- `cs-bg-is-white`

## Example
```html
<div class="cs-bg-is-white">
    ...
</div>
```
