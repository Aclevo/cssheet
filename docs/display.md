# Display
The following classes change the display of the element:

- `cs-display-block`
- `cs-display-flex`
- `cs-display-inline-block`
- `cs-display-none`
- `cs-display-grid`
- `cs-display-inline` 
- `cs-display-inline-grid` 
- `cs-display-flow-root`


## Example
```html
<a class="cs-display-block">
    ...
</a>
```
