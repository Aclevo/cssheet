# Foreground Color
There are a few different classes that you can add to your HTML elements in order to change their foreground color, one for each of the colors in the variable section:
- `cs-is-primary`
- `cs-is-secondary`
- `cs-is-yellow`
- `cs-is-red`
- `cs-is-orange`
- `cs-is-purple`
- `cs-is-grey`
- `cs-is-black`
- `cs-is-white`

## Example
```html
<p class="cs-is-primary">
    ...
</p>
```
